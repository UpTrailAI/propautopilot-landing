# PropAutopilot Landing Page

Marketing site + waitlist capture for PropAutopilot.

## Quick Start
```bash
pnpm install
pnpm dev        # → http://localhost:3000
```

## Deploy
```bash
vercel deploy
```

## Specs
- `LANDING-PAGE-BRIEF.md` — Full page spec (sections, copy, design, SEO)
- `CLAUDE.md` — Instructions for Claude Code

## Built With
Next.js 15 · TypeScript · Tailwind CSS · Framer Motion · shadcn/ui
